# v0.2.19

## Bug Fixes

- **Archived/shipped releases display inconsistently in sidebar and detail (#143)** — Collapses the three diverging release labels ("Published" from GitHub, "Shipped" from Tiki, blank for archived) into a single "Completed" display state. v0.2.16–v0.2.18 now show the same Completed badge as older releases, both in the sidebar and the detail panel. The fix lives at the rendering layer only — `#142`'s decision to hide archived releases from `load_tiki_releases` is preserved, and the Rust `TikiReleaseStatus` enum still distinguishes `Completed` from `Shipped` for any future logic that needs the difference.

**Full Changelog**: https://github.com/Eric-Ness/Tiki-V2/compare/v0.2.18...v0.2.19
