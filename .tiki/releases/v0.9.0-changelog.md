# v0.9.0

**Tiki Doctor** ‚Äî a read-only `.tiki/` workspace health check, surfaced in the desktop app. Detects drift like the stale-"active" release class (#259) in-app instead of by eye.

## Features

- **Tiki Doctor backend: `tiki_doctor` command** (#261) ‚Äî a read-only Tauri command returning a `DiagnosticsReport`: framework version, `state.json` validity + schema version, active-work count, per-release location/status/`archivedButActive` checks, `recentReleasesMissingJson` history‚ÜîJSON parity gaps, and reconciler-hook detection. Pure inspection ‚Äî mutates nothing.
- **Tiki Doctor UI: Settings ‚Üí Diagnostics panel** (#262) ‚Äî renders the `tiki_doctor` report with pass / warn / info indicators and a Refresh button. Warnings fire only for genuine drift (invalid state, missing release JSON, absent reconciler hook); the normal archived-but-active state is shown as neutral info, not a warning.
