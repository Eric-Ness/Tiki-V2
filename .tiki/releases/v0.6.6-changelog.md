# v0.6.6

First feature release surfaced from the 2026-05-20 codebase deep-dive — three additions that activate dormant data-model capacity and stand up the long-specified extensibility layer.

## Features

- **Live planning checklist** (#216) — PLAN's success criteria become a live checklist: EXECUTE marks each criterion verified once **all** of its covering phases (per `coverageMatrix`) complete, and the detail panel renders a ticking ☑/☐ list with `N/M` progress for both in-flight and completed issues. Activates the previously-dormant `SuccessCriterion.verified`/`verifiedAt` fields plus a pure `deriveCriteriaVerification` helper — **no schema change**.

- **config.json schema, validation, and in-app editor** (#214) — `.tiki/config.json` gains a canonical JSON schema (`additionalProperties:false` at every level, covering `workflow.tests`/`autoHeal`/`parallel`, `changelog`, and `backupRetention`), a `validateConfig()` that flags misspelled keys as **warnings instead of silently defaulting**, new atomic `read_tiki_config`/`save_tiki_config` Tauri commands, and a Settings → Workflow editor with inline validation. Closes backlog item E33.

- **Lifecycle hooks** (#215) — implements the long-specified (`DESIGN.md` §"Lifecycle Hooks", never built) `.tiki/hooks/` system as a real `run-hook.mjs` runner invoked from EXECUTE and SHIP. Six hook points (`pre`/`post-execute`, `phase-start`/`phase-complete`, `pre`/`post-ship`) with documented `TIKI_*` env vars; `.ps1` (Windows) and `.sh` (Git Bash) support; `pre-*` hooks block the pipeline on non-zero exit, `post-*`/`phase-*` warn only. Sample registry ships with all hooks disabled. See [`docs/HOOKS.md`](../../docs/HOOKS.md).

## Internal

- `@tiki/shared`: +21 tests (checklist verification + config validation).
- desktop: +25 vitest cases (criteria checklist + workflow-config helpers); +6 Rust tests (config parsing / unknown-key collection).
- framework: new `run-hook.mjs` with 11 `node:test` cases (suite 10 → 21).
- Test totals at ship: shared 86, desktop 144, framework 21, Rust 34. `pnpm build` + `cargo clippy --deny warnings` clean.
