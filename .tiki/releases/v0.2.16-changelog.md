# v0.2.16

## Features

- **Inline plan editor in desktop app** ([#116](https://github.com/Eric-Ness/Tiki-V2/issues/116)) — Plans can now be modified directly from the desktop app's issue detail panel without re-running `/tiki:plan` or hand-editing JSON. Edit phase fields, reorder phases via drag-and-drop, add/remove phases and success criteria, with automatic coverage matrix recomputation, structural validation, and a before/after JSON diff before save.
