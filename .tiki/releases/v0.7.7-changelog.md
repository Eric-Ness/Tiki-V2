# v0.7.7

## Bug Fixes
- **Terminal: bottom prompt row stays visible after output/resize** (#254) — the terminal could leave its bottom row (the shell / Claude Code "Enter text here" prompt) painted below the viewport until a keystroke refreshed it. The output path now re-pins the viewport to the bottom after writes (guarded so scrolling up to read history isn't interrupted), and every resize path routes through a `fit → scrollToBottom → refresh` sync with an RAF-debounced `ResizeObserver`. Also removed a redundant, leaked window-resize listener.

## Features
- **`reconcile-state.mjs --print` reconciler doctor** (#253) — a read-only command that prints each active issue's recorded-vs-derived pipeline step and flags drift, for diagnosing pipeline-tracking freezes. (Already on `main` since the v0.7.5/v0.7.6 reconciler verification; tagged here.)
